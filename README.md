#Spring3 MVC Tutorial

  This repository is an example web application for spring mvc tutorial.


###Getting Started:

- go to project directory
- run command

~~~
 mvn jetty:run
~~~

- URL: 

~~~
 http://localhost:8080/projectspring
~~~
